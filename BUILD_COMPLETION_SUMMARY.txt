================================================================================
TROUBLESHOOTING
================================================================================

If Xcode shows "Product not found" errors:
  1. Run: npx cap sync ios
  2. In Xcode: Product → Clean Build Folder
  3. Rebuild: Product → Build (Cmd+B)

If SPM package resolution is slow:
  First build may take 2-5 minutes (normal)
  Xcode will show progress
  Subsequent builds are instant

If location/camera permissions don't work:
  1. Verify Info.plist has permission keys (✅ already done)
  2. Simulator: Features → Location → set location
  3. Device: Settings → App → Permissions
