WHERE DATA APPEARS IN THE APP:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📍 TRAINING PAGE (TrainingPage.tsx)
   └─ Calendar view shows your activity
   └─ Queries: external_activities (includes GPS)
   └─ Displays: Name, distance, duration, sport icon
   └─ Marks source as: "Asciende GPS"

📊 PERFORMANCE DASHBOARD (PerformanceDashboard.tsx)
   └─ Heatmap (84-day calendar)
   └─ Queries: external_activities + athlete_workouts + extra_training_logs
   └─ Shows: Days you were active (color intensity = activity count)
   └─ Calculates: Weekly totals, frequency stats

🗺️ ACTIVITY HISTORY PAGE (ActivityHistoryPage.tsx)
   └─ Dedicated page for LOCAL GPS activities
   └─ Queries: activities table (not external_activities)
   └─ Shows: Complete activity details + map
   └─ Filters: By sport type, by date range
   └─ Map displays: Your route from GPS points

📱 ATHLETE DASHBOARD (AthleteDashboard.tsx)
   └─ Today's activities section
   └─ Queries: external_activities for today
   └─ Shows: Quick summary of activity

HOW DATA FLOWS FROM PHONE TO DISPLAY:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

YOU RECORD ACTIVITY
       ↓
ActivityRecorder.tsx captures GPS every second
       ↓
You press "Save Activity"
       ↓
save-activity Edge Function executes
       ├─ INSERT → activities table (1 row)
       ├─ INSERT → activity_gps_points table (~1800 rows)
       └─ INSERT → external_activities table (1 row)
       ↓
SUCCESS! Data now visible in:
       ├─ TrainingPage (within seconds)
       ├─ PerformanceDashboard (within seconds)
       ├─ ActivityHistoryPage (within seconds)
       └─ AthleteDashboard (within seconds)
